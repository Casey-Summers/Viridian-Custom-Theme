# Viridian

The main colours of the theme are:

**Primary** - `#c8ffea`<br>
**Seconday** - `#9ad99e`<br>
**Buttons** - `#2ecc71`<br>
**Footer** - `#005f5f`<br>


## Site logo
The theme requires the use of a site logo for the header and navigation, to add/change the site logo go to Appearance -> Customize -> Site identity
